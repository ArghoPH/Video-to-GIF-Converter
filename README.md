# Video-to-GIF-Converter
# 🎬 Video to GIF Converter

<div align="center">

![Video to GIF Banner](https://img.shields.io/badge/Video-GIF%20Converter-22c55e?style=for-the-badge)

A modern **Video to GIF Converter** built with **HTML + TailwindCSS + JavaScript**.

🔥 Convert videos into high-quality GIFs directly in your browser.  
⚡ Drag & Drop support.  
✂️ Custom crop editor.  
📦 No server required.

</div>

---

# ✨ Features

- 🎥 Convert Video → GIF
- 🖱️ Drag & Drop Upload
- ✂️ Free Custom Crop
- 🟩 Square Crop Preset
- 📺 16:9 Crop Preset
- 📥 Download Generated GIF
- ⚡ Local Browser Processing
- 🔒 No Upload to Server
- 🎨 Beautiful Tailwind UI
- 📱 Responsive Design
- 🚀 Works Offline After Load

# 🚀 How to Run

## 1. Clone Repository

```bash
git clone https://github.com/ArghoPH/Video-to-GIF-Converter.git
```

---

## 2. Open Project

```bash
cd Video-to-GIF-Converter
```

---

## 3. Run with Live Server

You can use:

- VS Code Live Server
- XAMPP
- Any local server

# 🎮 Usage

## Upload Video
- Click **Choose Video**
OR
- Drag & Drop directly

---

## Crop Video
- Resize crop box
- Move crop area freely

---

## Generate GIF
- Select start/end time
- Choose output width
- Click **Generate GIF**

---

## Download
- Preview generated GIF
- Click **Download GIF**

---

# ⚠️ Heavy Video Support

✅ Supports large videos (4GB+)  
⚠️ Recommended GIF duration:

| Type | Recommended |
|---|---|
| Meme GIF | 2–4 sec |
| Cinematic Clip | 4–6 sec |
| Long Scene | 6–8 sec |

---

# 💡 Best Settings

| Setting | Recommended |
|---|---|
| Width | 720 / 1080 |
| Quality | Balanced |
| Duration | Under 8 sec |
